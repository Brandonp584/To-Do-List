# 🚀 Full Stack Task Manager

A modern full-stack task management application built with the MERN stack featuring secure authentication, responsive UI, inline Task editing, skelton loading states, toast notifications, and polished Saas-Style UX.

## ✨ Features

### 🔐 Authentication

- User registration & Login
- JWT authentication
- Protected routes
- Persistent sessions with localStorage
- Show / Hide password toggle
